# Glamorous
 A full scale front end website using Bootstrap 4


Tools used:
 - HTML/CSS
 - Bootstrap 4


Main Features:
  - Fully responsive to all screen sizes (using bootstrap and custom media queries)
  - Collapsible text areas
  - Sliding Carousels
  - Expandable images
  - Video play button
  - Multi-paged
  - Overall great look and feel to the site
